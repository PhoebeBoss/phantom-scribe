# Phantom Scribe

Journal agent for Phantom Capital. Raw, first-person, anti-slop writing. Powers void-mirror entries.

## Tech Stack
- TypeScript, Express
- Docker + Zeabur
- tsx runtime

## Development
```bash
npm install
npm run dev    # Watch mode
npm run start  # Production
```

## Deployment
Deployed on Zeabur via `zbpack.json` + Dockerfile.
